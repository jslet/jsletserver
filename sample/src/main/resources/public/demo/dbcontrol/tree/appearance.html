Jslet - 树控件样式

缺省样式树（适用树节点等高的场景）

无连接线的树（适合树节点不等高的场景）

箭头样式的树（适合做树形菜单等）

演示内容：

演示树形控件用于各种场景。

源码

HTML

    //缺省样式树（适用树节点等高的场景）
    <div data-jslet="type: 'DBTreeView', dataset: 'agency',expandLevel: 0" style="height: 200px"></div>

    //无连接线的树（适合树节点不等高的场景）
    <div data-jslet="type: 'DBTreeView', dataset: 'agency1',expandLevel: 0, hasLine: false" style="height: 200px"></div>

    //箭头样式的树（适合做树形菜单等）
    <div data-jslet="type: 'DBTreeView', dataset: 'agency2',expandLevel: 0, useArrow: true" style="height: 200px"></div>

JavaScript


	var dsAgency = jslet.data.getDataset('agency');
	
	//创建第二个数据集
	var dsAgency1 = dsAgency.clone('agency1');
	dsAgency1.records(dsAgency.records());
	
	//创建第三个数据集
	var dsAgency2 = dsAgency.clone('agency2');
	dsAgency2.records(dsAgency.records());

	jslet.ui.install();
