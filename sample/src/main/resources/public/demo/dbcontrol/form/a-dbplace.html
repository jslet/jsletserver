Jslet - DBPlace

DBPlace为一个占位控件，实际的编辑控件来自于字段对象的editControl属性。

源码

HTML

    //姓名，部门，生日三个编辑控件，除了字段名不一样外，其它完全一样
    <div data-jslet="type: 'DBPlace', dataset: 'employee', field: 'name'"></div>
    <div data-jslet="type: 'DBPlace', dataset: 'employee', field: 'department'"></div>
    <div data-jslet="type: 'DBPlace', dataset: 'employee', field: 'birthday'"></div>
    
    //工资（带前后缀部分）
    <div data-jslet="type: 'DBPlace', dataset: 'employee', field: 'salary', prefix: [{content: '税后', width: '30px'}], suffix:[{field: 'currency', width: '80px'}]"></div>
JavaScript

	//datasetMetaStore定义在公共js:common/datasetmetastore.js中
	//将数据集定义信息仓库加到datasetFactory中，创建Dataset时会仓库里去定义信息
	jslet.data.datasetFactory.addMetaStore(window.datasetMetaStore);
	//通过工厂方法，可以自动创建主数据集及相关的数据集
    jslet.data.datasetFactory.createDataset('employee').done(function() {
    	jslet.data.getDataset('employee').query();
    	jslet.ui.install();
    });
