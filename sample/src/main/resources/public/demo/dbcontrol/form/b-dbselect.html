Jslet - DBSelect

DBSelect可用于带lookup的字段，也可用于非lookup字段（需自行设置option元素），DBSelect在显示大数据时有性能问题，可用DBComboSelect或者DBAutoComplete代替。

源码

HTML

    //非lookup字段（“姓名”字段）
    <select data-jslet="type: 'DBSelect', dataset: 'employee', field: 'name'">
		<option>Name1</option>
		<option>Name2</option>
		<option>Name3</option>
	</select>
    
    //带lookup字段（“部门”字段）
    <select data-jslet="type: 'DBSelect', dataset: 'employee', field: 'department'"></select>
    
    //带分组功能（“部门”字段按“地址”分类）
    <select data-jslet="type: 'DBSelect', dataset: 'employee', field: 'department', groupField:'address'"></select>
    
    //多选（“部门”字段）
    <select multiple data-jslet="type: 'DBSelect', dataset: 'employee', field: 'department'" style="height: 100px"></select>
    
    //多选并且带分组功能（“部门”字段按“地址”分类）
    <select multiple data-jslet="type: 'DBSelect', dataset: 'employee', field: 'department', groupField:'address'" style="height: 100px"></select>
JavaScript

	//datasetMetaStore定义在公共js:common/datasetmetastore.js中
	//将数据集定义信息仓库加到datasetFactory中，创建Dataset时会仓库里去定义信息
	jslet.data.datasetFactory.addMetaStore(window.datasetMetaStore);
	//通过工厂方法，可以自动创建主数据集及相关的数据集
    jslet.data.datasetFactory.createDataset('employee').done(function() {
    	jslet.data.getDataset('employee').query();
    	jslet.ui.install();
    });
