Jslet - 表格高度自动伸缩

演示内容：

一般情况，表格的高度应该固定，以免造成界面跳动。但在一些紧凑的录入界面上，也需要表格能够根据数据量自动伸缩，以免界面上有空白区域。

表格控件有3个属性用于表格的自动伸缩：autoStretch， minVisibleRows，maxVisibleRows。

点击“删除”或者“新增”，观察表格高度的变化。

源码

HTML

    <div data-jslet="type: 'DBTable', dataset: 'employee', autoStretch: true, minVisibleRows: 1, maxVisibleRows: 5"></div>
Javascript

	//datasetMetaStore定义在公共js:common/datasetmetastore.js中
	//将数据集定义信息仓库加到datasetFactory中，创建Dataset时会仓库里去定义信息
	jslet.data.datasetFactory.addMetaStore(window.datasetMetaStore);
	//通过工厂方法，可以自动创建主数据集及相关的数据集
    jslet.data.datasetFactory.createDataset('employee').done(function() {
		var dsEmployee = jslet.data.getDataset('employee');
		dsEmployee.query();
    	jslet.ui.install();
    });
